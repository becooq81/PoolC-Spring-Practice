# PoolC-Spring-Practice

풀씨 (동아리) 서버와 유사하지만 주요기능에 집중하여 개선한 서버 사이드 프로젝트입니다. 

### 주요 기능 목록
- 회원가입, 로그인 및 회원수정
- 유저 등급 분리 및 관리
- 동아리 회원 확인 기능
- 세미나 등록, 신청, 수정 기능
- Naver API를 사용하거나 직접 입력으로 책 등록, 책 대출 및 반납 기능
- AWS를 이용한 CI/CD 프로세스 구축
- Github Action을 이용한 Pre-hook, Post-hook

### 사용 기술

- Spring Boot
- MySQL
- AWS 배포
